## How to use 

Firstly, install the dependencies using `pip`

```
pip install tk tkcalendar pillow datetime
```

Go to the root of the project and run:

```
python tela.py
```

### Funcionalities

* Search for register
* Delete register
* Create register
* Update register
* Upload/Change photo
* See data on table
